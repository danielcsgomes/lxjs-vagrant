# LXJS Vagrant

This Repository is part of the workshop **[Building a Baby PaaS](http://2013.lxjs.org/workshops#docker)** at **[LXJS 2013](http://2013.lxjs.org/)**.

## Instalation

 1. git clone git@github.com:danielcsgomes/lxjs-vagrant.git
 2. cd lxjs-vagrant
 3. vagrant up
 4. vagrant ssh
